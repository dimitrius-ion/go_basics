# go_basics

## go structure

workspaces -> modules -> packages -> files

## Create a new workspace

```bash
mkdir workspace_name
cd workspace_name
```

## Create a new module

```bash
go mod init github.com/username/repo
```

## Run a go file

```bash
go run main.go
# or
go run .
```

## crete a package

```bash
mkdir package_name
touch package_name/package.go
```
